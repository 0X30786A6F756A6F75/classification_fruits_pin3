# Trabalho PIN 3
## Instalação (DEV)
--
Necessário:
Python
PIP
--
1. Criar ambiente virtual
```bash
$ python3 -m venv env
```

2. Roda o ambiente virtualizado
```bash
$ source env/bin/activate
```

3. Instalar Dependecias:
```bash
$ pip install -r requirements.txt
```

4. Rodar o projeto
```bash
$ python train.py
```
