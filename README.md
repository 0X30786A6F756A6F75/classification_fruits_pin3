# Roda back
1. Caminhe até o projeto de back
```bash
$ cd back
```
2. Ative o ambiente virtual
```bash
$ source env/bin/activate
```
3. Instalar depedências
```bash
$ pip install -r requiremets.txt
```

4. Rodar projeto
```bash
$ flask run
```
---
# Roda front
1. Caminhe até o projeto de front
```
$ yarn install
```
2. Rode o projeto 
```bash
$ yarn start
```
